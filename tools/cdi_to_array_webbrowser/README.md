
Web Browser based tool to converts a valid Configuration Description Information (CDI) 
XML file into a C structure section to paste into the node_parameters.c file in 
the structure initializer.

Double click on the xml2cdi.html file, select a XML file and press "Convert" press "Copy to Clipboard" and paste the code into your node_parameters.c file replacing any 

  .cdi = {
     0x3C, 0x3F, 0x78, 0x6D, 0x6C,.....

     .....
   },

section that is there.
