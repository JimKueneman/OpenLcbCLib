Implements a FIFO of core buffers for CAN messages that hold CAN only frame messages that need to be passed on to the nodes. This buffer is accessed in the CAN Rx statemachine and the main loop so using Pause and Resume to stop the Rx and 100ms timer when accessing it is critical.
