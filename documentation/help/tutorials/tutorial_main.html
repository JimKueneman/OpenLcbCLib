Welcome to the OpenLcbCLib Tutorial Page

OpenLcbCLib is a library written in the C programming language. It uses very few special feature of the language and should be compatible with a wide selection of compilers. The library makes creating OpenLCB or NMRA LCC firmware that is compliant to the networks standards simple and reduces cycle time as the application level firmware does not need to know much about the networks standards and can focus on core features the node implements.

Some of the features of the library are:

  • Supports features of the OpenLCB standards and the NMRA LCC subset of these standards

  • Uses dependance injection to help keep the modules independent and fully testable through Google Test scripts

  • Uses a consistent model of how tasks and state machines and state machine handers are implemented
