Detailed Description

User-editable TCP/IP driver configuration for OpenLcbCLib.

OPTIONAL: If this file is not present the TCP driver will use built-in defaults. Copy this file to your project's include path and edit to override any value.

All values use #ifndef guards in tcp_types.h so defining them here (or via -D compiler flags) takes priority over the library defaults.
