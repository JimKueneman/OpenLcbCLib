◆ TcpTxStatemachine_send_openlcb_message()

bool TcpTxStatemachine_send_openlcb_message ( openlcb_msg_t * msg)

Converts an OpenLCB message to TCP wire format and transmits it.

Builds the 17-byte preamble followed by MTI, source Node ID, optional destination Node ID, and payload bytes. Calls the user's transmit_raw_tcp_data() callback with the complete byte sequence.

This function is wired as the transport's send_openlcb_msg() in the OpenLCB main state machine interface when OPENLCB_COMPILE_TCP is defined.

Parameters
msgPointer to the message to send. The caller retains ownership of the buffer (this function does not free it).
Returns
true on success, false if TX buffer is busy or transmit fails.

Sends raw bytes with a TCP/IP OpenLCB preamble (link control).

Used by TcpLinkControl to send link-level messages (status, drop link). The flags field has bit 15 clear (not an OpenLCB message).

bodyPointer to the link control body bytes.
