Detailed Description

Platform-specific mDNS backend interface.

Users provide their platform's mDNS implementation via these function pointers. All are REQUIRED.

Example implementations:

  • ESP32: esp_mdns_query_ptr / mdns_service_add
  • Linux: avahi_client / avahi_service_browser
  • macOS: DNSServiceBrowse / DNSServiceRegister
