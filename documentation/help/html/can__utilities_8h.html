uint16_t CanUtilities_extract_dest_alias_from_can_message ( can_msg_t * can_msg)
extern

Extracts any destination Alias ID from the CAN frame buffer.

Note
This function examines the frame and pulls out the destination Alias based on what type of message it is. If the message does not carry a destination Alias it return 0.
Parameters
can_msg_t*can_msg_source - Pointer to a CAN frame buffer with the identifier.
Returns
The destination Alias extracted from the CAN identifier or 0 if there is no destination carried in this frame
