Detailed Description

Bit flags for Protocol Support Inquiry/Reply messages.

The Protocol Support Reply message contains a 6-byte (48-bit) bit field indicating which optional protocols the node implements. Bits are numbered with bit 0 as the LSB of byte 0.

Required protocols (always supported):

  • Protocol Support Protocol itself
  • Simple Node Information Protocol (SNIP)
See also
MTI_PROTOCOL_SUPPORT_INQUIRY
MTI_PROTOCOL_SUPPORT_REPLY
