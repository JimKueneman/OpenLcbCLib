void TcpLinkControl_handle ( uint16_t flags,
const uint8_t * data,
uint16_t len )
extern

Handles an incoming link control message.

Called by the RX state machine when a message with flags bit 15 = 0 is received. Routes to the appropriate handler based on the link control type in the body.

Parameters
flags16-bit flags from the preamble.
dataPointer to the link control body bytes (after preamble).
lenNumber of body bytes.

◆ TcpLinkControl_run()

bool TcpLinkControl_run ( tcp_statemachine_info_t * info)
extern

Drives the link control run-loop.

Retries any pending Status Reply or Drop Link Reply. For Drop Link Reply, fires on_link_drop_requested only after the reply has been successfully sent.

Parameters
infoPointer to the TCP statemachine info (carries pending bits).
Returns
true if work is still pending, false if idle.
