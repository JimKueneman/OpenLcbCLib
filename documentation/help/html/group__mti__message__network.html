Detailed Description

Message Type Indicators for basic node communication.

These MTI codes handle fundamental node operations:

  • Node initialization and identification
  • Protocol capability discovery
  • Error reporting and interaction rejection
See also
OpenLCB Message Network Standard S-9.7.1.1
