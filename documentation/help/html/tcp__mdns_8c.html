bool TcpMdns_discover_hub ( tcp_mdns_result_callback_t callback)

Queries for OpenLCB hub services on the local network.

Starts an mDNS query for "_openlcb-hub._tcp.local.". Results are delivered asynchronously via the callback.

Parameters
callbackFunction called for each discovered hub.
Returns
true if the query was started, false on error.
