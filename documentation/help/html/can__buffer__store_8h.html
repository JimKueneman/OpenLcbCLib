Detailed Description

Implements the core buffers for CAN frames.

The CAN FIFO buffer is an array of pointers to CAN frame buffer ( can_msg_t) The are allocated and freed through function calls CanBufferStore_allocate_buffer() and CanBufferStore_free_buffer().

Note
Applications typically have no reason to access these buffers directly, they should use the helpers to send messages in openlcb_application.h
Warning
The CAN Receive Statemachine and 100ms timer access these buffers and typically run within interrupts and/or threads. Care must be taken to Pause and Resume the interrupts or threads if the main loop needs to access the buffers for any reason.
