Detailed Description

Constants for managing node enumeration keys.

Nodes can be enumerated using user-defined keys (0-3) and internal system keys (4-6). This allows different subsystems to independently enumerate nodes.

User keys (0-3): Available for application use Internal keys:

  • Key 4: Main state machine enumeration
  • Key 5: Login state machine enumeration
  • Key 6: CAN state machine enumeration
