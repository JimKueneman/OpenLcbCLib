Detailed Description

Implements a state machine actually a large switch statement that decodes the incoming message and calls the correct the can_tx_message_handler.h handler functions.

This is the core transmit functionality of the CAN frames. It either sends a CAN message directly, which is a direct translation to the CAN physical layer or sends an OpenLcb/LCC message which may require being broken up into multiple CAN frames.
