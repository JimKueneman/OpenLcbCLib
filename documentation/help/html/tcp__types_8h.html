Analogous to can_types.h for the CAN transport layer.

◆ TCP_PREAMBLE_LEN

#define TCP_PREAMBLE_LEN   17

Maximum bytes in the RX accumulation buffer.

Override at compile time: -D USER_DEFINED_TCP_RX_ACCUMULATION_BUFFER_LEN=2048

Maximum bytes in the TX scratch buffer.

Override at compile time: -D USER_DEFINED_TCP_TX_BUFFER_LEN=2048

Maximum concurrent multi-part message reassemblies.

Override at compile time: -D USER_DEFINED_TCP_MAX_MULTIPART_ASSEMBLIES=4

Total preamble length: 2 flags + 3 length + 6 originating NodeID + 6 capture time.

◆ tcp_statemachine_info_t

typedef struct tcp_statemachine_info_struct tcp_statemachine_info_t

Context block passed through the TCP state machine on every iteration.

Carries the current node pointer and control flags. Analogous to can_statemachine_info_t for the CAN driver.

tcp_main_statemachine.h

◆ tcp_multipart_entry_t

typedef struct tcp_multipart_entry_struct tcp_multipart_entry_t

One slot in the multi-part message reassembly table.

Keyed by originating Node ID from the preamble. Accumulates payload bytes across first/middle/last parts until a complete OpenLCB message can be forwarded to the protocol layer.
