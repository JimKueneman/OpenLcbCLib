Detailed Description

Configuration memory read handler — processes datagram read commands for CDI (0xFF), All (0xFD), Config (0xFE), ACDI-Mfg (0xFC), ACDI-User (0xFB), Train FDI (0xFA), and Train Fn Config (0xF9).
