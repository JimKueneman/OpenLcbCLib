Detailed Description

Instruction bytes and sub-commands for Train Control Protocol.

Byte layouts for MTI_TRAIN_PROTOCOL (0x05EB) commands and MTI_TRAIN_REPLY (0x01E9) replies.

See also
OpenLCB Train Control Standard S-9.7.4.6
