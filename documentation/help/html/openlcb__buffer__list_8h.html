Detailed Description

Implements a Linear search buffer list for OpenLcb/LCC message structures.

A buffer list that can hold OpenLcb/LCC message structures. Used to hold messages that are coming in using a physical layer adaptation of the OpenLcb/LCC
message where it can be sent in 1 or many frames. This buffer can hold partially received messages until they are fully assembled.

Note
Applications typically only need to access the Initialize function in this module.
Warning
The physical layer receive state machines and 100ms timer access these buffers and typically run within interrupts and/or threads. Care must be taken to Pause and Resume the interrupts or threads if the main loop needs to access the buffers for any reason.
