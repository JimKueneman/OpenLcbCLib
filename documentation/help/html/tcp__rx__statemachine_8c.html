◆ TcpRxStatemachine_incoming_data()

void TcpRxStatemachine_incoming_data ( uint8_t * data,
uint16_t len )

Feeds incoming TCP data into the receive state machine.

Called from the user's socket receive callback or thread. Accumulates bytes, parses preambles, and pushes complete OpenLCB messages to the FIFO. May be called with any number of bytes (partial messages are buffered internally).

Parameters
dataPointer to received bytes.
lenNumber of bytes received.
Warning
May be called from an interrupt context or a dedicated receive thread. Uses lock_shared_resources/unlock_shared_resources around buffer allocation and FIFO push.
