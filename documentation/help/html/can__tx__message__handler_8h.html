Detailed Description

Implements handlers for use by the CAN Transmit (Tx) statemachine can_tx_statemachine.h.

As CAN and OpenLcb/LCC messages are transmitted these handlers transmit different types of messages.

Note
Applications typically only need to access the Initialize function in this module.
Any handler may be overridden by assigning a custom function pointer to the interface_can_tx_message_handler_t field during initialization of the application. see: CanTxMessageHandler_initialize().
