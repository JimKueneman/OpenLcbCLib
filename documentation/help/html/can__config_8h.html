void CanConfig_initialize ( const can_config_t * config)
extern

Initializes the CAN bus transport layer.

Must be called BEFORE OpenLcb_initialize().

Parameters
configPointer to can_config_t configuration. Must remain valid for the lifetime of the application.
Warning
NOT thread-safe - call during single-threaded initialization only.
See also
openlcb_config.h

Algorithm:

  1. Save the user-provided config pointer.
  2. Initialize buffer infrastructure (CanBufferStore, CanBufferFifo).
  3. Build all 7 internal interface structs from user config and library functions.
  4. Initialize all CAN modules in dependency order: RxMessageHandler, RxStatemachine, TxMessageHandler, TxStatemachine, LoginMessageHandler, LoginStateMachine, MainStatemachine, AliasMappings.
* @param config  Pointer to @ref can_config_t configuration. Must remain
*                valid for the lifetime of the application.
*
Warning
NOT thread-safe - call during single-threaded initialization only.
