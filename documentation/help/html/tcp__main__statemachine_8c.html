◆ TcpMainStatemachine_run()

bool TcpMainStatemachine_run ( void )

Drives the TCP main state machine run-loop.

Calls the login run-loop during LOGGING_IN state, and the link control run-loop to flush pending replies.

Returns
true if work is pending, false if idle.
